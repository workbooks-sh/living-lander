05 · the weave

From a sentence to a living page

You write what you want. The agent weaves it from toolkits. The runtime renders it — and keeps it running. Three moves, and the artifact is alive.

01
you write the intent

a plain sentence — what the thing should be and do. Not a spec, not code. A description in your own words.

02
the agent weaves

it reaches into toolkits for the capabilities it needs and composes them into a working artifact — a page, a tool, a flow.

03
it runs, and it keeps running

the runtime renders the artifact and holds it open — editable like a document, alive as long as it's running. This page is one. You're reading it work.
